# Crunch Website

GitHub-Pages-fertige Landing Page für Crunch.

## Veröffentlichen
1. ZIP entpacken.
2. Neues GitHub-Repository erstellen.
3. Alle Dateien inklusive `assets` hochladen.
4. `Settings` → `Pages`.
5. `Deploy from a branch`, Branch `main`, Ordner `/root`.

## Waitlist
Das aktuelle Formular ist eine Demo und speichert noch keine E-Mails. Ersetze den `<form>`-Block später durch den Embed-Code von Tally oder Kit und aktiviere Double-Opt-in.

## Vor Veröffentlichung
- Impressum vervollständigen
- Datenschutz anpassen
- Kontakt-E-Mail ergänzen
- Testeintragung durchführen

## App-Screenshot austauschen
Ersetze `assets/crunch-app-screenshot.png` durch einen neuen Screenshot mit demselben Dateinamen.
